## Solution

–ö–∞–∂–¥—ã–π —Å–∏–º–≤–æ–ª –≤ `ASCII` –º–æ–∂–µ—Ç –ø—Ä–∏–Ω–∏–º–∞—Ç—å –∑–Ω–∞—á–µ–Ω–∏–µ –æ—Ç `0` –¥–æ `255`, –∑–Ω–∞—á–∏—Ç –≤ `hex` —ç—Ç–∏ –∑–Ω–∞—á–µ–Ω–∏—è –±—É–¥—É—Ç –æ—Ç `0x00` –¥–æ `0xFF`. –†–∞–∑–æ–±—å—é —Å—Ç—Ä–æ–∫—É –ø–æ 2 —Å–∏–º–≤–æ–ª–∞, –ø–µ—Ä–µ–≤–µ–¥—É –∏—Ö –≤ –¥–µ—Å—è—Ç–∏—á–Ω—ã–µ —á–∏—Å–ª–∞, –∞ –¥–∞–ª–µ–µ –ø–æ–ª—É—á—É —Å–∏–º–≤–æ–ª `ASCII`. 

–í–æ—Ç –∫–æ–¥ –¥–µ–∫—Ä–∏–ø—Ç–æ—Ä–∞ –Ω–∞ `Python`:

```python
s = '63727970746f7b596f755f77696c6c5f62655f776f726b696e675f776974685f6865785f737472696e67735f615f6c6f747d'

flag = []

for i in range(0, len(s), 2):
    c = s[i:i+2]
    c = int(c, 16)
    flag.append(chr(c))

print(''.join(flag))
```
